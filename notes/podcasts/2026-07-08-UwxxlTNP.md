# The 100,000 Sandbox Problem — Akshat Bubna, Modal CTO

KEY INVESTABLE INSIGHTS

The primary trend is the emergence of "Agent Experience" (AX) as the next evolution of "Developer Experience" (DX). This suggests a new layer of infrastructure is being built to serve AI agents as the primary user, abstracting away complexities like Kubernetes. Platforms that provide simple, secure, and ephemeral execution environments ("sandboxes") for AI agents will be critical. This creates a new market for "self-provisioning runtimes" and could disrupt incumbent DevOps and cloud-native tooling companies that don't adapt.

AI/TECH SECTOR TRENDS

- **Infrastructure Mismatch for AI:** The podcast highlights that existing cloud infrastructure, particularly Kubernetes, is not optimized for the "bursty," resource-intensive, and short-lived compute tasks common in AI (inference, batch jobs, training). This creates an opportunity for new, specialized cloud platforms.

- **Developer Experience to Agent Experience:** The core thesis is that the principles of a good developer experience (simplicity, abstraction) now apply to AI agents. Agents should not have to write or manage complex configuration files (e.g., YAML for Kubernetes). Instead, they need a "self-provisioning runtime" that can be controlled programmatically through simpler interfaces.

- **GPU Demand Inflection:** The guest's experience confirms that while GPU demand for "classical" machine learning (computer vision, etc.) existed pre-2023, the generative AI boom created a massive and distinct acceleration in demand for GPU-based inference.

COMPETITIVE DYNAMICS

- **Hyperscaler Commoditization Risk:** Platforms that successfully abstract away infrastructure complexity could reduce the "stickiness" of the big three cloud providers (AWS - AMZN, Azure - MSFT, GCP - GOOGL). By offering a superior developer/agent experience that runs on top of any cloud, these platforms turn the underlying hyperscaler into a more commoditized compute provider.

- **New Layer in the Stack:** The need for specialized AI infrastructure and agent-native platforms indicates a new layer of value creation is forming on top of the foundational cloud providers. Companies building these primitives (e.g., for elastic inference, sandboxing, training) are in a position to capture significant value.

CAPITAL ALLOCATION & GPU SUPPLY

- **Capacity Management is a Differentiator:** The ability to manage a multi-cloud, multi-region pool of GPUs and other compute resources is a key business advantage. This involves sophisticated forecasting of both supply (from various cloud providers) and demand to deliver cost-effective and reliable performance for customers. There is a compute supply shortage that requires active management.
